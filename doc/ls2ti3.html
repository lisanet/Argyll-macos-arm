profile/ls2ti3

Summary

Convert LightSpace format RGB .bcs test chart results into Argyll .ti3 CGATS format.

Usage Summary

usage: ls2ti3 [-v] infile outbase
 infile        Input LightSpace .bcs file
  outbasefile   Base name for output.ti3 file

Usage Details and Discussion

ls2ti3 takes a LightSpace  .bcs test chart results, and converts them into Argyll .ti3 CGATS files.
